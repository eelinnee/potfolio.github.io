Веб-сайт кинотеатра

Цель: разработать веб-сайт кинотеатра с совеременным дизайном, который упрощает процесс выбора фильмов и бронирования билетов.

  • Удобный формат расписания сеансов
  • Подробная информация о фильмах и актерах
  • Система бронирования мест в кинозале
  • Новостная система с актуальной информацией о мире кино

[ главная страница ]

Была создана главная страница с удобной навигацией по сайту с сохранением общей цветовой гаммы, чтобы не перегружать визуал. На главном экране представлены акционные предложениям и премьеры недели, данные элементы были разработаны и воплощены в AdobePhotoshop.

[ расписание сеансов ]

При нажатии на ссылку расписания сеансов открывается окно с расписаниями на ноябрь-декабрь. Была разработана многоуровневая система навигации, где наиболее востребованные элементы размещены на боковой панели и в шапке для максимальной доступности. Продуманный интерфейс календаря, где цветовое кодирование активных дней позволяет пользователю мгновенно определить доступные для бронирования даты, сокращая время на принятие решения.

[ сеансы на день ]

Детальная страница сеансов с интерактивными элементами: кликабельные постеры фильмов открывают трейлеры для принятия осознанного решения, а система цветового кодирования времени сеансов обеспечивает визуальную обратную связь при наведении, улучшая навигацию.

[ схема зала ]

Адаптивная система выбора мест: в зависимости от спроса на фильм и времени сеанса пользователь попадает в оптимальный по размеру кинозал. Цветовая дифференциация мест позволяет быстро ориентироваться в доступных вариантах без визуальной перегрузки.

[ оформление покупки билета ]

Процесс оформления билета с визуальными подсказками и валидацией данных — система автоматически проверяет корректность ввода и уведомляет о необходимости дополнения информации перед подтверждением покупки.

Приложение-справочник по Вселенной Гарри Поттера

Цель: разработать мобильное приложение на Android Studio по Вселенной Гарри Поттера с представлением всесторонняя информация о различных факультетах Хогвартса, таких как Гриффиндор, Слизерин, Хаффлпафф и Рейвенкло, чтобы пользователи смогли узнать об отличительных свойствах учеников каждого факультета, его

  • Удобная навигация по разделам приложения
  • Подробная информация о персонажах и факультетах
  • Полный список заклинаний с описанием
  • Система карточек с кратким обзором и детальными страницами
  • Опитимизация приложения путем извлечения информации через API

[ экран загрузки приложения ]

Разработан экран загрузки (Splash Screen) в тематике Вселенной Гарри Поттера — анимированая волшебная палочка и примечание в виде заклинания из Гарри Поттера, "открывающего все двери".

[ главный экран приложения ]

Начальный экран, содержащий набор карточек, через которые можно переходить на страницы факультетов для подробной информации о них, и две карточки, посвященные заклинаниям и экранизации. Также внизу находится панель навигации (BottomNavigationView). Был проведен тщательный анализ, чтобы подобрать наиболее подходящие вектор-изображения под стиль приложения.

[ экранная форма факультетов ]

Одна из экранный форм приложения по факультетам. Для других страниц факультетов информация представлена в схожем стиле, только в соответствующей факультету цветовой тематике и информации.

[ экранная форма заклинаний ]

Данная экранная форма представлена в виде списка заклинаний, извлеченных через соответствующий API. Она извлекает наименование заклинания и краткое описание к нему.

[ экранная форма экранизаций ]

Данная экранная форма представлена также в виде списка, но содержит в себе — изображение и название фильма, дату выпуска, его длительность и крактое описание. Вся информация также извлекается через соответствующий API.

[ экранная форма волшебников ]

Одна из экранных форм списка волшебников, информация о которых извлечена из API. Здесь также извлекаются изображения и краткая информация о волшебнике, для отсутствующих фотографий прилагается фотография, созданная самостоятельно в AdobePhotoshop с припиской 'ImageNotFound'

Оптимизация работы сайта на Moodle

Цель: улучшить визуальное представление элементов курса и навигацию по сайту.

  • Удобный формат навигации по сайту
  • Улучшенное представление элементов курса
  • Простая, удобная и красивая система в карточке курса

[ главная страница ]

Ранее на главной странице были представлены элементы курсов. Была выполнена цель — исправить визуальное представление на главной странице. Теперь здесь размещено крактое описание организации с ссылкой на все элементы предоставляемых ими услуг с возможностью перехода на данный список курсов. Интуитивная навигация с подсветкой элементов направляет к детальному каталог курсов, улучшая пользовательский опыт. Кроме того, было реализовано слайд-шоу, визуализирующее спектр услуг через тематические изображения, а также добавлена карта с указанием расположения данной организации.

[ элемент курса ]

Проведена комплексная оптимизация карточек курсов: расширено описание с добавлением целей и направленности обучения, внедрена система кликабельных элементов для каждого раздела с краткими аннотациями содержания, а адаптивные изображения улучшили визуальное восприятие и навигацию по учебным материалам.

[ элемент раздела курса ]

Улучшенная навигация по элементу раздела данного курса. Включает в себя кликабельные элементы для перехода на лекцию с её кратким описанием.

[ итоговое тестирование ]

Улучшен интерфейс тестирования: добавлена информация о количестве вопросов и баллов, тематическое изображение и мотивационное сообщение для лучшего пользовательского опыта и снижения тревожности перед тестом.
