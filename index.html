Приложение-справочник по Вселенной Гарри Поттера

Мобильное приложение на Android Studio по Вселенной Гарри Поттера. В приложении представлена всесторонняя информация о различных факультетах Хогвартса, таких как Гриффиндор, Слизерин, Хаффлпафф и Рейвенкло. Пользователи смогут узнать об отличительных свойствах учеников каждого факультета, его характеристиках и основных элементах, связанных с ними. Это поможет фанатам лучше понять, какие черты и ценности отличают студентов каждого факультета. Кроме того, Wizardly содержит обширные списки заклинаний, которые исследуют магические способности персонажей.

  • Удобная навигация по разделам приложения
  • Подробная информация о персонажах и факультетах
  • Полный список заклинаний с описанием
  • Система карточек с кратким обзором и детальными страницами
  • Опитимизация приложения путем извлечения информации через API
